# Le terminal

## Définition

Le terminal est une interface en ligne de commande qui permet d'interagir avec le système d'exploitation en exécutant des commandes textuelles.

Il est utilisé pour naviguer dans les dossiers, manipuler des fichiers, lancer des programmes et exécuter des outils de développement.

## Commandes de base

| Commande | Description                                           |
| -------- | ----------------------------------------------------- |
| `pwd`    | Affiche le répertoire courant.                        |
| `ls`     | Liste les fichiers et dossiers du répertoire courant. |
| `cd`     | Change de répertoire.                                 |
| `mkdir`  | Crée un nouveau dossier.                              |
| `touch`  | Crée un fichier vide.                                 |
| `cp`     | Copie un fichier ou un dossier.                       |
| `mv`     | Déplace ou renomme un fichier.                        |
| `rm`     | Supprime un fichier ou un dossier.                    |
| `clear`  | Nettoie l'écran du terminal.                          |

## Pourquoi utiliser le terminal ?

* Exécuter rapidement des commandes.
* Utiliser Git.
* Installer des dépendances avec Node.js et npm.
* Automatiser certaines tâches de développement.
