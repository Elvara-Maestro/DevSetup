## Description

**DevSetup** est le premier projet du parcours **Fullstack Akieni Academy**.

Ce dépôt a été créé afin de mettre en place un environnement de développement complet et de valider l'installation ainsi que la configuration des principaux outils utilisés dans le développement logiciel.

Il sert également de documentation personnelle regroupant des notes sur les outils installés et les commandes essentielles.

---

## Outils installés

* Git
* GitHub
* Node.js
* Visual Studio Code (VS Code)

---

## Structure du projet


.
├── README.md
├── .gitignore
├── .editorconfig
└── notes
    ├── git.md
    ├── installation.md
    └── terminal.md


---

## Contenu du dépôt

### README.md

Présente le projet, son objectif et son organisation.

### .gitignore

Contient les fichiers et dossiers qui ne doivent pas être suivis par Git.

### .editorconfig

Définit des règles communes de mise en forme du code (indentation, fin de ligne, encodage, etc.) afin de garantir une présentation cohérente des fichiers, quel que soit l'éditeur utilisé.

### notes/

Ce dossier regroupe des notes de travail rédigées durant la mise en place de l'environnement de développement.

* **git.md** : principales commandes Git et notions de base.
* **installation.md** : étapes d'installation et de configuration des outils.
* **terminal.md** : commandes essentielles du terminal.

---

## Comment utiliser ce dépôt

1. Cloner le dépôt :

```bash
git clone https://github.com/<ELVARA-MAESTRO>/DevSetup.git
```

2. Accéder au projet :

```bash
cd DevSetup
```

3. Consulter les différents fichiers de documentation présents dans le dossier **notes**.
