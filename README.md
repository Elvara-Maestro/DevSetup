# Environnement de développement

## Auteur
**Bénit Christ-Veille MAMONSONO**

## Description

Ce dépôt contient la configuration de mon environnement de développement réalisée dans le cadre du programme de formation **Akieni**.

Il a pour objectif de confirmer à l'équipe pédagogique que tous les outils nécessaires au développement ont été correctement installés et configurés.

### Outils installés

- ✅ Git
- ✅ Node.js
- ✅ Visual Studio Code (VS Code)
- ✅ GitHub

## Structure du dépôt

Le dépôt est organisé comme suit :

```
.
├── README.md
├── .gitignore
└── notes
    ├── git.md
    ├── installation.md
    └── terminal.md
```

### Contenu

- **README.md** : Présentation générale du projet.
- **.gitignore** : Liste des fichiers et dossiers à ignorer par Git.
- **notes/** : Documentation personnelle sur les outils utilisés.
  - **git.md** : Notes sur Git et les commandes essentielles.
  - **installation.md** : Étapes d'installation des outils.
  - **terminal.md** : Commandes de base du terminal.

## Objectif

Ce dépôt constitue le premier livrable du parcours de formation Akieni et atteste que mon environnement de développement est opérationnel pour la réalisation des prochains projets.
